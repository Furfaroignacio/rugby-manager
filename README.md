# Rugby Manager

Sistema de gestión deportiva para equipos de rugby.

## Stack
- **Backend:** Python + FastAPI + PostgreSQL
- **Frontend:** React + Vite + TailwindCSS
- **Auth:** JWT

## Estructura
- `/backend` — API REST con FastAPI
- `/frontend` — SPA con React

Ver instrucciones en `/backend/README.md` y `/frontend/README.md`
